HTML Tags

Paragraph tag <p> </p>
<p> tag defines a paragraph and we can use for paragraph.
Heading tags <h1> to <h6>
<h1> is the most important heading, and <h6> is the least important.
Emphasis tag <em> </em>
We can use this tag to emphasize the word or paraphrase, and the content inside is displayed in italic. Also, the screen reader will emphasize the word.
Highlighting tag <mark> </mark>
We can highlight the text with this tag.
Quote tag <q> </q>
<q> tag defines a short quote. We can use for short quotes.
